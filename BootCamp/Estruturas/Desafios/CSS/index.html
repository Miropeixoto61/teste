Agrupamento de Seletores

Podemos aplicar as mesmas regras CSS para varios seletores diferentes. pPAra isso, podemos separar os seletores utiliando uma virgula(,).

Assim, não precisamos ficar copiando e colando os mesmos códigos para todos, nós podemos fazeer um só bloco de CSS e reutilizar asmesmas regras!
